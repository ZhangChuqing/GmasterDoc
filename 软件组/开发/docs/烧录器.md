# 烧录器
## 烧录线
标准线序为绿黄黑红,分别对应3v3，GND，CLK，DIO

## stlink
没啥好说的，插好线在flash.cfg内将烧录器改好即可使用。


## CMSIS-DAP
### 设置模式

长按按键A，直到指示灯变为黄色。短按按键B，根据指示灯颜色调节为需要的模式。

• 如果需要将配置保存到Flash中（即下次上电保持当前配置）：长按按键A，直到指示灯变为黄色闪烁，再短按按键A退出。

• 如果只是临时使用，不保存配置到Flash：直接短按按键A退出。
### 设置无线通信

长按按键B，直到指示灯变为闪烁状态。此时闪烁颜色和次数，表示当前所属WiFi信道：

• 绿色 → 1-5 信道

• 红色 → 6-10 信道

• 蓝色 → 11-13 信道

短按按键B选择目标信道后，长按按键A直到指示灯变为黄色闪烁，再短按按键A退出。

⚠️ 主机和从机需要分别设置成一样的信道才可以通信，完成后主机和从机需要重新上电。
### 无线模式
#### 配对

 1. 第一个模块按住按键B上电，直到指示灯变为紫色再松手。

 2. 第二个模块按住B上电，它会在指示灯变紫后，瞬间两个模块分别变为主机（蓝色）和从机（绿色）。

此时两端已经自动设置好通信的参数和模式，如果需要可以通过按键进行更改。完成后主机和从机需要重新上电，在flash.cfg中将烧录器改为cmsis-dap即可使用。
