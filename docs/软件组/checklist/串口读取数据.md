# 串口读取数据
按以下顺序排查：
1. **是否检测到端口**  
   - 若没有，换一根数据线  
   - 换另一个 USB 口（A 或 C 口）  
   - 尽量不要使用扩展坞  

2. **端口是否选择正确**  
   - 确认软件中选择的 COM 口和设备管理器一致  

3. **驱动是否正确安装**  
   - 打开设备管理器  
   - 若端口有黄色警告图标，右键更新/安装驱动  

4. **波特率是否一致**  
   - 发送端与接收端的波特率必须一致  

5. **TX / RX 是否接反**  
   - **TX 接 RX，RX 接 TX**  

---

如果仍无法读到数据，请记录现象并联系老登协助排查。
