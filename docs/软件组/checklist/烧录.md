# 烧录出错

 1. 终端报错：openocd 找不到

    **现象：**
    ```bash
    'openocd' 不是内部或外部命令，也不是可运行的程序
    ```

    **原因：**
    - 终端红色报错如下-->环境没配置好，将openocd路径加入环境变量
    ```bash
    ‘openocd’不是外部命令，也不是可运行的程序
    ```

    **解决方法：**
    - 将openocd路径加入环境变量
    ```bash
    export PATH=$PATH:/path/to/openocd
    ```

 2. 烧录线序是否正确
    
 3. 无线烧录器是否配对

    参考开发文档[烧录器](../开发/烧录器.md#2-cmsis-dap-无线烧录器)

 4. 无线烧录器没有上电烧录

 5. 确认**flash.cfg**中烧录器类型正确

 6. 更换 USB 线 / USB 口

 7. 仍然无法正常烧录

    怀疑硬件，更换C板和烧录器。
