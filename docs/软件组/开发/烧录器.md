# 烧录器

## 烧录线序
标准线序为 **绿-黄-黑-红**，对应关系如下：

- **绿** → 3.3V (3V3)
- **黄** → GND
- **黑** → CLK
- **红** → DIO

> ⚠️ 连接前请确认目标板电压为 3.3V，避免损坏设备。

---

## 1. ST-Link

ST-Link 配置很简单：

1. 按正确线序连接烧录线  
2. 在 `flash.cfg` 中将烧录器类型设置为 **stlink**
3. 直接烧录即可

## 2. CMSIS-DAP 无线烧录器

### 2.1 设置模式

1. **长按按键 A**，直到指示灯变为黄色（进入模式设置）  
2. **短按按键 B**，根据指示灯颜色选择目标模式  
3. 退出方式：  
   - **保存配置到 Flash**：长按 A 直到黄色闪烁，再短按 A 退出  
   - **临时使用**：直接短按 A 退出  
### 2.2 设置无线通信信道

1. **长按按键 B**，直到指示灯开始闪烁  
2. 闪烁颜色与次数表示当前信道范围：  
   - **绿色** → 1–5 信道  
   - **红色** → 6–10 信道  
   - **蓝色** → 11–13 信道  
3. **短按 B** 切换信道  
4. 设置完成后，**长按 A** 直到黄色闪烁，再**短按 A** 退出  

> ⚠️ 主机与从机需设置为**相同信道**，完成后需**重新上电**。
### 2.3 无线模式配对

1. **第一个模块**按住 **B** 上电，直到指示灯变为紫色再松手  
2. **第二个模块**按住 **B** 上电，指示灯变紫后，两端会自动分配：  
   - **主机** → 蓝色  
   - **从机** → 绿色  

此时通信参数已自动配置完成，如需修改可按键调整。完成后**主机与从机需重新上电**。  

最后在 `flash.cfg` 中将烧录器改为 **cmsis-dap** 即可使用。
