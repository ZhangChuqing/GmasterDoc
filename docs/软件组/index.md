# 软件组知识库

哈喽！欢迎来到软件组的知识仓库，这里是我们用挠头换经验的地方。

无论你是刚入坑的萌新，还是身经百战的“老油条”，这里都塞满了我们一路走来踩过的坑、填上的土，以及偶尔灵光一闪的解法。希望它们能帮你少掉几根头发，多写几行能跑的代码。

但有一件比什么都重要的事，必须说在最前面：

⚠️ 不要无脑相信 AI，更别把代码全扔给 Copilot 写！​ ⚠️

它也许能给你一段看起来不错的代码，但它不会告诉你为什么这样写、边界情况是什么、在实车上会不会抽风。真正理解逻辑、控制细节、应对突发状况的，永远是你自己的脑子。AI 可以是你的“搜索引擎plus”，但不能成为你的“大脑离线版”。

所以请多问为什么，多试几次，多看看车跑起来的样子。

这里的所有文档，都是为了让你成为那个真正在控制代码的人，而不只是复制粘贴的搬运工。

代码可以自动生成，但解决问题的能力不会。

咱们一起，保持思考，保持调试。
