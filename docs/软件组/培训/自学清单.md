# 自学清单

## 重要性说明

- 必修：必须学习和观看的内容，需按课程计划完成观看学习，将设置quiz进行考核，课上会进行讲解和答疑。
- 辅修：建议学习的内容，作为对必修知识的补充，但不设置考核，且课上不进行讲解。
- 选修：可选学习的内容，作为兴趣拓展和深入研究的材料，仅供自我提升。
