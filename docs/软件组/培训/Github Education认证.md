
# GitHub Education 认证教程

## 写在前面

本教程用于 GMaster 战队培训的补充教程，尽可能使教程更加适合西浦学生。

GitHub Education 包含 GitHub 平台为学生与教职工提供的一系列重要权限与资源。其中的重要免费权限之一：**Copilot Pro**，将为认证者提供大量的 AI 使用额度，极大助力代码开发工作。

## 教程正文

## 认证前的准备

### Microsoft Authenticator

**介绍**

Microsoft Authenticator 是微软推出的免费 APP，用于提供更安全的账户登录方式。本教程将使用该软件完成 GitHub 的双重验证（2FA）。

**下载方式**

- **iOS**：App Store 搜索 *Microsoft Authenticator*
- **Android**：应用商店 / Google Play 搜索下载

**其他准备**

- 注册并登录微软账户  
- 注册 GitHub 账户  
- 准备学生卡照片  

---

## 认证方式

## 一、为 GitHub 绑定 edu 邮箱

### 1. 打开 Settings

右上角头像 → **Settings**

### 2. 添加学生邮箱

进入 **Emails** → 在 *Add email address* 中填写你的学生邮箱，例如：

```
Zhang.San25@student.xjtlu.edu.cn
```

点击 **Add**

### 3. 输入 GitHub 密码验证

系统提示：

> We sent a verification email to xxx@student.xjtlu.edu.cn.

### 4. 打开学生邮箱，点击邮件中的 **Verify email address**

### 5. 打开 GitHub 注册邮箱，复制收到的验证码

### 6. 输入验证码 → 点击 **Verify**

### 7. 成功添加 edu 邮箱

---

## 二、绑定 2FA 双重验证

### 1. 打开 Two-Factor Authentication

路径：

```
Settings → Password and authentication → Two-factor authentication
```

点击 **Enable two-factor authentication**

### 2. 使用 Microsoft Authenticator 扫描二维码

按照网页提示：

- 打开手机端 Microsoft Authenticator
- 扫描 GitHub 提供的二维码
- 输入 App 中生成的 6 位验证码

完成绑定。

---

填写示例：

| 字段 | 内容 |

| City | Suzhou Jiangsu Province |
| State/Province | Jiangsu Province |

## 四、进行 GitHub Education 认证

> **注意：请不要使用科学上网，确保 IP 在学校区域。**

### 1. 打开 Education Benefits

路径：

```
Settings → Education Benefits → Start an application
```

选择：

- **Student**
- 学校：Xi'an Jiaotong-Liverpool University（若自动识别）
- 学校邮箱：填写你的 `@student.xjtlu.edu.cn`

若提示 **Share Location**，请允许定位。

如无法定位，请使用手机浏览器打开 GitHub 网页端进行认证。

---

### 2. 上传学生卡照片

选择：

```
Dated school ID
```

上传学生卡照片 → 点击 **Submit Application**

> 若无法上传，请使用电脑摄像头拍摄学生卡。  
> 若仍无法通过，请确保在校园网环境下提交。

---

### 3. 提交成功（Pending 状态）

你会看到：

```
Pending
Your application has been received and is currently pending review.
```

---

### 4. 审核通过（Approved 状态）

稍后你会收到：

```
Approved
Your academic status has been verified. Congratulations!
```

并提示：

- 72 小时内开通所有学生权益
- 可领取 Copilot Pro 优惠券
- 权益有效期一般为 2 年
